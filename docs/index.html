Donots

Send macOS notifications to email.

Download for macOS Requires macOS 14.0 or later •
🔔

Notification Monitoring

Watches the macOS Notification Center and triggers on every incoming alert.

📸

Region Capture

Automatically captures a high-quality screenshot of the notification area.

📧

Email Automation

Seamlessly sends the captured image via Apple Mail to your specified recipient.

🎯

Visual Selection

Includes a crosshair tool to precisely select the capture region on your screen.

⚙️

Adjustable Quality

Configure JPEG/PNG compression, scale, and quality to balance detail and size.

📊

Activity Logging

Monitor events in real-time with a dedicated log window and menu bar controls.

Quick Setup

  1. Open Donots and navigate to Settings > Permissions.
  2. Grant Accessibility, Screen Recording, and Mail Automation permissions.
  3. Configure your recipient email address in the main window.
  4. Use the crosshair tool to select your notification capture area.
  5. Click Start and you're ready to go!
